# Telegram Advisor Bot

## How to install?
## 1-step

```bash
npm install
```
## 2-step
### Create a new .env file and paste your telegram bot token on varaible called BOT_TOKEN (you can see a sample .emv.sample)

## 3-step
```bash
npm run dev
```
